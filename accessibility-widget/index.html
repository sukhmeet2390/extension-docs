Frequently asked questions

Does this make my store fully WCAG compliant?
The widget adds an accessibility toolbar for visitors to customize their experience. Full WCAG compliance requires additional code-level fixes, but the widget covers the most common usability barriers.
Will it slow down my store?
No. The widget loads asynchronously and is under 20KB. Zero impact on Core Web Vitals.
Is it free?
Yes. The core accessibility toolbar is free. Install from the Shopify App Store.
